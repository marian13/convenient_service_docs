Introduction

#StandWithUkraine

Convenient Service

Yet another approach to revisit the service object pattern in Ruby, but this time focusing on the unique, opinionated, moderately obtrusive, but not mandatory features.

* Logo is downloaded from CuteWallpaper.org. It will be replaced later.

WARNING ❗❗❗​

This library is under heavy development. Public API may be subject to change. The first major release is still to come. Use the current version at your own risk. Thanks.

Features​

Common Features​

Common features (with slight variations) compared to existing alternatives.

  • JSend and dry-monads inspired Results.

  • Steps similar to Trailblazer Operation Wiring.

  • Active Model Attributes integration influenced by ActiveAttr (optional).

  • Active Model Validations integration like in ActiveInteraction (optional).

Unique Features​

Features specific to this gem.

  • The menu is omakase.

  • Simple, predictable, and powerful at the same time plugin-based customization.

  • One-time delayed Concerns and Middlewares in the core.
